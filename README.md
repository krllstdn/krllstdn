### Selected projects

**Deep Archetypal Analysis of Kidney Donors** — *Master's thesis*
Deep archetypal contrastive autoencoder for kidney donor quality assessment from gene expression. Extends classical archetypal analysis with supervised contrastive learning using donor clinical features as side information. Pretrained on ~14k public GEO samples, fine-tuned on 276 IKEM biopsies. Publication in preparation.
`TensorFlow · contrastive learning · autoencoders`

**[Local RAG Pipeline](https://github.com/krllstdn/rag)**
Fully local retrieval-augmented generation over a personal knowledge base: chunking, embedding, vector retrieval and generation with self-hosted LLMs.
`Ollama · Qdrant · LangChain · FastAPI`

**[Protein Function Prediction](https://github.com/krllstdn/protein-function-prediction)**
Protein function annotation from raw sequence with several deep learning architectures (CNNs, Inception CNN, LSTMs, Transformers) on UniProt data. Best model (Inception CNN) AUC = 0.808.
`TensorFlow · deep learning · bioinformatics`

**[Anomaly Detection in Medical Imaging](https://github.com/krllstdn/ae-anomaly-detection)**
Autoencoder-based unsupervised detection of dental caries in X-rays (97.4% accuracy, synthetic data). Comparative study of Classical AE, Convolutional AE and VAE with interpretability heatmaps.
`TensorFlow · autoencoders`

**[Exon Search](https://github.com/krllstdn/ExonSearch)**
CNN approach to identifying exon sequences in the human genome. 78.3% accuracy on a balanced validation set.
`CNNs · TensorFlow · BioPython`

**[Gene Finding](https://github.com/krllstdn/GeneFinding)**
Hidden Markov Model (42-state) with Viterbi decoding for E. coli gene identification. 74% Jaccard index on test sequences.
`HMM · Viterbi · genomics`

**[Kidney Transplant Survival Analysis](https://github.com/krllstdn/BachelorProject)** — *Bachelor's thesis*
Survival models (Random Survival Forests, Cox, Gradient Boosted Survival Trees) for post-transplant outcome prediction, with an end-to-end pipeline and a React + Django REST application.
`survival analysis · scikit-learn · Django · React`
